Conceitualização

Uma função qudrática é aquela que tem uma única variável com a forma:

f(x)=ax2+bx+c, a ≠ 0;

Onde x é sua variável e a, b e c são seus coeficientes.

O gráfico dessa função é uma parábola.

Se a função for igualada a 0, então obtemos uma equação quadrática.

Suas soluções são as raízes (ou zeros).

Uma equação quadrática pode ter duas, uma ou nenhuma solução.


Sinais do coeficiente a

Obs: Se a > 0, a parábola abre para cima



Obs2: Se a < 0, a parábola abre para baixo



Resolvendo uma Equação Quadrática

Para resolver uma função quadrática, a igualamos a 0.

Exemplo: f(x)=3x2+4x+1

Começemos igualando a função a 0:

3x2+4x+1=0

Para resolvermos essa equação, usamos Bhaskara:

Ela se dá por:

x=-b±b2-4.a.c2.a

Substituíndo a, b e c na fórmula temos:

x=-4±42-4.3.12.3 =-4±16-126 =-4±46 =-4±26

x₁=-4+26 =-26 =-13

x₂=-4-26 =-66 =-1

Esses dois valores de x são os zeros, ou seja, nossa resposta.



Determinante de uma Equação

O disciminante é o Delta (Δ). Dado que a fórmula de bháskara pode ser escrita como:

x=-b±Δ2.a

Δ=b2-4ac

Sua função é determinar o número e natureza das raízes.

Existem 3 possiblidades:

Quando Δ>0

A equação possui 2 raízes reais e distintas


Quando Δ=0

A equação possui 2 raízes reais e iguais


Quando Δ<0

A equação possui duas raízes complexas.

Nesse caso em especial, a equação não tem resposta dentro dos números reais.

Ou seja, ela envolve números imaginários, sendo este o "i" que equivale a √-1

Isto ocorre porque não exite raiz quadrada de números negativos.



Intersecção de duas funções

A intersecção de duas funções se dá por sua igualdade.

Ou seja, são os pontos em comum das funções quadráticas f(x) e g(x).

Exemplo: f(x) = x2+x+1 e g(x) = -x2+2x+4

Primeiro encontramos os valores de x dos pontos ao igualarmos as funções.

x²+x+1 = -x²+2x+4 → 2x²-x-3 = 0

Agora resolvemos a equação usando bháskara:

x=-(-1)±(-1)2-4.(-3).22.2 =1±1-(-24)4 =1±254 =1±54

x₁=1+54 =64 =32

x₂=1-54 =-44 =-1

Sabemos agora que x= -1 e 3⁄2.

Então, subtituímos esses valores em uma das equações para encontrar o y

f(-1) = (-1)2+(-1)+1=1-1+1 = 1

f(3⁄2) = (3⁄2)2+(3⁄2)+1 = 32⁄22 + 3⁄2+1 = 9⁄4+6⁄4+4⁄4 = 19⁄4

Portanto, os pontos de intersecção de f(x) e g(x) são: (-1,1) e (3⁄2,19⁄4)



Vértice de uma função

O vértice de uma função é encontrado usando a seguinte fórmula:

f(x) = a(x-(-b⁄2a))2+(c-b2⁄4a)

Onde -b⁄2a é o ponto em x e c-b2⁄4a é o ponto em y

Exemplo: f(x)= x2+2x-1

Usando a fórmula,temos:

x = -2⁄2.1 = -1

y = (-1)-22⁄4.1=-1-4⁄4 = -1-1 = -2

Portanto, o vértice é (-1,-2)



Deslocamento de uma função

O deslocamento se dá quando comparamos o vértice com um um ponto.

Ou seja, o deslocamento em x é x1-x2 e o deslocamento em y é y1-y2

Exemplo1: f(x) = x2+2x-1. Encontre o deslocamento de f(x) em relação a origem (0,0)

Como essa é a mesma função do exemplo do vértice, podemos pular direto para a comparação.

Deslocamento em x = (-1)-0=-1

Deslocamento em y = (-2)-0=-2

Portanto, f(x) se deslocou -1 unidade em x e -2 unidades em y

Esboço de um gráfico

O esboço do nosso gráfico será feito calculando e conectando vários pontos.

Seja f(x) = x2-4x+3

Encontrando os Zeros

Resolvendo f(x) = 0

x2-4x+3 = 0

x=-(-4)±(-4)2-4.1.32.1 =4±16-122 =4±42 =4±22

x₁=4+22 =62 =3

x₂=4-22 =22 =1

Portanto, os zeros são: x=1, 3

Encontrando a intersecção com y

f(0) = 3

intersecção com y é em y = 3

Encontrando o vértice

x = -(-4)⁄2.1 = 4⁄2 = 2

y = 3-(-4)2⁄4.1 = 3-16⁄4 = 3-4 = -1

Portanto, o vértice é em (2, -1)

Esboço

Sabemos agora todos os nossos pontos: (3, 0), (1, 0), (0, 3), (2,-1). Então é só conectar.
