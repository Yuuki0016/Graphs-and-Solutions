Conceitualização

Funções Lineares são aquelas cujo gráfico é uma linha reta, ou seja, é uma função polinomial de grau zero ou 1.

Também conhecida como função afim, tem a fórmula f(x)=ax+b, onde a e b são constantes.

O gráfico de tal função é uma linha reta, onde a é o coeficinete angular e b é o coeficiente linear.

Se a>0, a função é crescente:



Se a<0, a função é decrescente:



Nesse contexto, uma função constante também é linear, sendo um polinômio de grau 0 e, gráficamente, uma reta horizontal



Sabendo o conceito, vamos nos aprofundar nos cáculos.



Determinação dos Coeficientes Angular e Linear

Para Determinar o coeficiente angular, usamos uma razão entre um ponto e outro.

A fórmula se dá por: a=y2-y1⁄x2-x1

Exemplo:

Dado o gráfico na imagem, percebemos os pontos (0,4), (1,7) e (2,10).

Usando a fórmula, podemos encontrar o coeficiente angular.

Utilizando dos pontos (0,4) e (1,7), encontramos o seguinte:

a= 7-4⁄1-0 = 3⁄1 = 3

Portanto, o coeficiente angular é igual a 3!

Mas e o coeficiente linear?

Bom, sabemos que agora f(x)=3x+b

Agora é só substituir um ponto aleatório da reta na fórmula!

Utilizaremos o ponto (2,10), portanto:

f(x)=3x+b → 10=6+b → b=10-6 → b=4

Logo, o coeficiente linear é igual a 4!

Então, temos nossa fórmula: f(x)=3x+4



Esboço de um gráfico

Tendo em mente a fórmula, como desenhamos?

Podemos apenas substituir um valor de x e encontrar o y, ou vice versa.

Ao executar algumas vezes, podemos ligar os pontos com uma reta.

Exemplo: f(x)=2x+3

Vamos substituír x com 0 e 2:

f(0)=2*0+3 → f(0)=3

f(2)=2*2+3 → f(2)=4+3 → f(2)=7

Agora só os ligamos.

Esse é o nosso esboço. Como gráfico é linear, o esboço já é o gráfico em si...



Resolvendo uma Função

Agora nós sabemos a fórmula e o y em um ponto, mas queremos descobrir o x.

Digamos que a função f(x)=3x-2 é igual a 7. Qual é o x?

É simples, substitua o valor de f(x) e resolva!

7=3x-2 → 7+2=3x → 9=3x → x=9⁄3 → x=3

Portanto, f(x)=7 quando x=3



Intersecção de Duas Funções

Para encontrar a intersecção de duas funções, apenas as igualamos.

Digamos que f(x)=3x+4 e g(x)=-4x+11

Vamos começar as igualando:

f(x)=g(x)

3x+4=-4x+11 → 3x+4x=11-4 → 7x=7 →x=7⁄7 → x=1

Descobrindo o x, é só substituílo em uma das funções para descobrir o y:

f(x)=3x+4 → f(1)=3*1+4 → f(1)=3+4 → f(1)=7

Portanto, f(x) e g(x) se encontram no ponto (1,7)


Note que, se as funções forem paralelas, não existe uma solução.



Domínio de uma Função

Um domínio se dá quando há um limite até onde a reta se estende.

Por exemplo:

f(x)=2x+3 (0 ≤ x ≤ 3)

Aqui, f(x) tem um domínio onde x não pode ser maior que 3 e nem menor que 0, ou seja, ele está entre 0 e 3.

Dado o domínio, vamos decobrir a imagem ao igualar x a 0 e 3:

f(0)=2*0+3 → f(0)=3

f(3)=2*3+3 → f(3)=6+3 → f(3)=9

Portanto, f(x)=2x+3 com um domínio onde x está entre 0 e 3, tem uma imagem onde f(x) está entre 3 e 9, ou seja, (3 ≤ f(x) ≤ 9)
