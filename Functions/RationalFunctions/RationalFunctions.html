Conceitualização

Uma função racional é aquela que tem uma razão.

Ou seja, uma divisão entre o numerador e o denominador, onde ambos são funções polinomiais

Ela tem a forma f(x) = P(x)⁄Q(x)

Domínio e Imagem

O domínio dessa função é dado para todos os valores de x que nao deixam o denominador ser igual a 0

Já a imagem é dada pelo fato de que apenas a razão 0⁄n resulta em 0, ou seja, valores fora da razão.

Exemplo: Seja f(x) = 2⁄x-1 +2

Neste caso, o domínio seria {x ≠ 1} e a imagem seria {y ≠ 2}

Note que a função aparenta tocar o eixo x e y fora do domínio, porém ela não está realmente tocando.

Esse é o limite, ou as assíntonas



Resolvendo Funções racionais

Para resolvermos funções racionais, efetuamos o mesmo processo de uma equação normal

Podemos aplicar o mmc para facilitar a resolução.

Exemplo: Seja f(x) = x2+3⁄x-1 e f(x) = 7

x2+3⁄x-1 = 7 → x2+3 = 7(x-1) → x2+3 = 7x-7 → x2-7x+10 = 0

Agora, resolvemos essa função quadrática

x=-(-7)±(-7)2-4.1.102.1 =7±49>-402/mrow> =7±9>2/mrow> =7±32/mrow>

x₁=7+32/mrow> =102/mrow> =5

x₂=7-32/mrow> =42/mrow> =2



Assíntonas

Assíntonas são basicamente um limite, ao quala função chega infinitamente perto, mas nunca se iguala.

Para encontrá-las, precisamos reduzir ao máximo a razão

Isto é: resolver a razão

Exemplo: Seja f(x) = 2x+1⁄x-1

2x+1⁄x-1 = 3⁄x-1 +2

Neste caso, as assintonas seriam: x = 1 e y = 2



Intersecção

A intersecção de duas funções se dá por sua igualdade, ou seja, f(x) = g(x)

Exemplo: Seja f(x) = 4⁄x+2 -1 e g(x) = -9⁄x-3 -2

4⁄x+2 -1 = -9⁄x-3 -2 → 4-x+2⁄x+2 = -9-2x+6⁄x-3

MMC = (x+2)(x-3)

-x-2⁄x+2 = -2x+3 → (x-2)(x-3) = (2x+3)(x+2) → x2-5x+6 = 2x2+7x+6

x2-5x+6-(2x2+7x+6) → x2-5x+6-2x2-7x-6 → x2+12x → x(x+12)

Portanto, a intersecção das funções ocorre em x = 0,-12

Substituindo em f(x):

f(0) = 4⁄0+2 -1 = 2-1 = 1

f(-12) = 4⁄(-12)+2 - 1 = 4⁄-10 -1 = -4⁄10 -10⁄10 = -14⁄10 = -7⁄5

Logo, os pontos de intersecção de f(x) e g(x) são: (-12, -7⁄5) e (0, -1)

Esboço

Para o esboço, apenas encontramos os pontos e os utilizamos.

Exemplo: Seja f(x) = x⁄x+2 -3

Encontrando as assíntonas

x⁄x+2 -3 = -2⁄x+2 -2

Assíntonas: x = -2 e y = -2

Encontrando as intersecções com x e y

f(0) = 0⁄0+2 -3 = 0-3 = -3

f(x) = 0, x⁄x+2 -3 = 0

x⁄x+2 - 3(x+2)⁄x+2 = x-3x-6⁄x+2 → -2x-6 = 0 → 2x = -6 → x = -3

Portanto, temos os pontos (0, -3) e (-3, 0), e asassíntoans x = -2 ey = -4
